# ZED2i Stereo Vision & Data Collection Project

This repository contains tools for working with the **StereoLabs ZED2i** and **Intel RealSense D435i** cameras. The project is divided into two main areas: synchronized data collection and stereo depth estimation using deep learning.

## Project Structure & Documentation

Detailed documentation for different components can be found in their respective directories:

### 1. [Stereo Image Processing & Depth Finding](stereoImageProcessing/code/StereoCameraFindingDepth/README.md)
Contains scripts for calculating real-world depth from stereo image pairs.
- **Key Features**: Mask R-CNN object detection, Hungarian algorithm matching, and centimeter-accurate depth estimation.
- **Location**: `stereoImageProcessing/code/StereoCameraFindingDepth/`

### 2. [Camera Data Collection Utilities](default/code/Working/README.md)
Core scripts for capturing synchronized video and IMU data from multiple cameras.
- **Key Features**: Parallel recording from ZED2i and RealSense D435i, versioned output folders, and IMU data logging (CSV).
- **Location**: `default/code/Working/`

### 3. [Specialized Data Acquisition](default/code/Working/samsang/collectionData/README.md)
Customized collection scripts for specific research or project requirements (e.g., the Samsung data collection project).
- **Key Features**: Highly configurable save options (`Color`, `IMU`, `Depth`), positional tracking, and multi-threaded performance optimizations.
- **Location**: `default/code/Working/samsang/collectionData/`

## Main Directories
- `default/`: Standard project workspace, containing core data collection logic.
- `stereoImageProcessing/`: Research and implementation of stereo-vision algorithms.
- `Data_RealSense/`: (Local) storage for recorded RealSense datasets.

## Requirements
Most tools in this repository require:
- **ZED SDK** (`pyzed`)
- **Intel RealSense SDK** (`pyrealsense2`)
- **OpenCV**
- **PyTorch & Torchvision** (for depth finding)
- **Pandas & NumPy**
